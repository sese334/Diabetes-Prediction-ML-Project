# Diabetes Prediction Project

## Overview
Machine Learning project to predict diabetes risk using medical and demographic data.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features
- Age
- BMI
- Hypertension
- HbA1c Level
- Blood Glucose Level

## Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Model Building
4. Evaluation

## Results
Achieved predictive analysis for identifying diabetes risk factors.
